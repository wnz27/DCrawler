### 练习内容: 珍爱网数据爬取
#### 总体算法
起始页面 --> 子页面  --> 去重
城市列表  --> 城市 --> 用户ssss
城市列表解析器 -》一堆城市 和url， 城市解析器 -》拿到一堆人和相应url
用户解析器 -》 拿到用户信息结构
#### 解析器 Parser
输入： utf-8 编码的文本
输出：Request{url, 对应parser}列表，item 列表
